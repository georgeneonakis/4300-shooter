CONTROLS:

WASD - Movement
Mouse - Aim
Left Mouse Button - Fire Weapon
Left Shift - Dash (while moving)
1, 2, 3, 4 - Cycle Weapons (must unlock 2, 3, and 4 by completing levels)
Z - Use Health Potion
X - Use Shield Potion
C - Use Speed Potion
V - Use Stealth Potion
Tab - View Inventory / Pause
Esc - Return to menu (quit if in menu)

Beat all 3 levels by collecting the weapons at the end to unlock the final boss!
